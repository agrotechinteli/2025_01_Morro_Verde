# User Stories

Lista histórias de usuário com base nas personas, seguindo o modelo "Como (persona), quero (necessidade) para (objetivo)".
User Stories

Persona 1: Mariana Silva (Gerente de Pricing)

1. Visão em tempo real de custo vs preço  
   “Como Gerente de Pricing, quero visualizar em tempo real o custo de produção e o preço de venda de cada tipo de fertilizante, para reagir imediatamente a variações de preço dos insumos e preservar a margem de lucro.”

2. Análise de sensibilidade de preço
   “Como Gerente de Pricing, quero simular diferentes cenários de preços e ver o impacto projetado na receita e na margem, para decidir rapidamente qual política de preços aplicar em cada região.”

3. Alertas automáticos de desvio de margem
   “Como Gerente de Pricing, quero receber alertas quando a margem de um produto cair abaixo de um patamar definido, para revisar imediatamente a política de preços ou renegociar compras de insumos.”

Persona 2: Carlos Oliveira (Analista de Vendas Regional)

1. Dashboard de tendência de demanda por região
   “Como Analista de Vendas Regional, quero ver um gráfico de tendências de demanda de fertilizantes por estado e cultura agrícola, para ajustar recomendações de estoque e planejar promoções locais.”

2. Comparação preço x volume vendido
   “Como Analista de Vendas Regional, quero comparar a evolução do preço médio praticado com o volume vendido semanalmente, para identificar pontos em que um preço alto começa a penalizar a venda.”

3. Notificações de oportunidades de upsell
   “Como Analista de Vendas Regional, quero ser notificado quando um produto com margem mais alta estiver com demanda crescente, para sugerir pacotes promocionais e aumentar o ticket médio.”
